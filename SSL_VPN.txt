SSL VPN Modes: Tunnel & web mode
  need a vpn client: forticlient
  web mode only needs a browser, but allows only limited protocols: ftp, http, https, RDP, SSH, Telnet, VNC, etc.

config vpn ssl web portal
  edit <portal-name>
  set tunnel-modes [enable|disable]
  set web-mode [enable|disable]
end

diagnose debug enable
diagnose vpn ssl <list/info/statistics/debug-filter/tunnel-test/web-mode-test>
diagnose debug application sslvpn -1
diagnose debug application fnbamd -1
diagnose debug console timestamp enable
